Perguntas frequentes

Posso hospedar no GitHub Pages?

Sim. Basta criar um repositório seu-usuario.github.io, adicionar este arquivo como index.html e ativar o Pages nas configurações.

Funciona sem internet?

Sim. O arquivo é único e não depende de bibliotecas externas. Ótimo para rodar localmente.

Como trocar as cores?

Altere as variáveis CSS em :root (--brand, --brand-2, etc.).
